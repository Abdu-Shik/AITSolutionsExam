# Airline Booking & Operations System

A full-stack application for managing airline operations and passenger bookings.

## Project Structure

This repository is divided into two main components:

- **[Backend (FastAPI)](./backend/README.md)**:
    - Handles API, Database (SQLite), and Authentication.
    - Located in `backend/` directory.

- **[Frontend (Flutter)](./frontend/README.md)**:
    - Mobile application for Passengers.
    - Located in `frontend/` directory.

## Quick Start

1.  **Start the Backend**:
    Follow the instructions in [backend/README.md](./backend/README.md).

2.  **Start the Frontend**:
    Follow the instructions in [frontend/README.md](./frontend/README.md).
